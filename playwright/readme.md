Playwright for Pythonのサンプルコード

以下リンクの内容を書き写したものです。
https://playwright.dev/python/docs/intro


## venvの作成

```shell
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## インストール

```shell
pip install pytest-playwright
playwright install
```

## テストの実行方法
```shell
pytest
```
